******************
* Run Instruction*
******************

* (RECOMMENDED) To run the NeruoNet Non-Linear Regression:
```
java -jar Group5Leader.jar nonlinear
```
* To run the NeuroNet Linear Regression
```
java -jar Group5Leader.jar linear
```

* To run the recurrent neuralnet LSTM
```
java -jar Group5Leader.jar lstm
```


if you want to see a debug information including profit gains run
```
java -jar Group5Leader.jar nonlinear --debug
```
